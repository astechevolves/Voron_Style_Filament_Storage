This cabinet started as a printer stand then became a place to put rolls to feed the tools and it just kept expanding. I am putting it all here to document and in case someone else is interested is using my lessons-learned for their own project. 


Notes to remember things to cover in repo:
ESP32 data to include temp and humidity sensors used
Filamentalist build and mounting
LLL Buffer modded firmware for I2C control (discuss other options tried and downfalls of each)
USB port, power port, and PTFE bulkheads (using a modded form of the StealthChanger frame)
Models to includes extrusion and brackets (different bracket options to make rigid)
Top and bottom covers to be .220 for rigidity for printer shelf as for caster mounting
Walls to be .118 PC because it was avaiable as local store
Door is just a larger clicky clack door with 3 hinges for consistancy
All walls are build the same way as the Voron using the same seals and clips
